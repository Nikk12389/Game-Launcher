# Game Launcher

A complete game launcher with 4 games integrated into one program.

## Games Included

1. Number Guessing Game
2. HangMan
3. Tic Tac Toe
4. Stone Paper Scissor

## How to Run

g++ GameLauncher.cpp -o launcher
./launcher

## How to Use

- Enter 1-4 to choose a game
- Enter 0 to exit launcher

## Controls for All Games

- 0 = Play again
- 1 = Exit to launcher
